正则表达式：符合一定规则的表达式
      作用：专门用于操作字符串
      好处：简化了对字符串的复杂操作
      特点：用一些特定的标签来表示一些代码操作。这样就简化了书写。
      弊端：符号定义越多，正则越长，阅读性越差。
      说明：其实String API中也有相应的对字符串的操作，但是操作起来比较复杂，所以可以使用正则表达式来简化操作。

      对于一些正则表达式的操作规则可以参考jdk的帮助文档


具体操作功能：
1.匹配：String 类的matches方法
2.切割：String类的split方法
3.替换：String类的replaceAll(String regex,String replacement)方法(把符合某规则的一部分替换成另外一部分)
4.获取：将字符串中符合规则的子串取出
获取的操作步骤：
    1，将正则表达式封装成对象。
    2，让正则对象和要操作的字符串相关联。
    3，关联后，获取正则匹配引擎。
    4，通过引擎对符合规则的子串进行操作，比如取出。
